mirror_server

ubuntu mirror server, ie: us.archive.ubuntu.com

mirror_proxy

proxy server to access the mirror_server (optional), ie: ‘http://proxy:3128/’

distro_version

ubuntu version, ie: xenial

root_pass

hashed root password

ntp_servers

list of ntp servers

packages

list of packages to install, ie: [ ‘open-vm-tools’ ]

postinstall_script

url to a script to download and executed in the target install, ie: http://static/stuff.sh

postinstall_commands

list of commands to run in the target install, ie: [ ‘echo “configured” > /etc/file’, ‘ls -l / > /stock_root’ ]

NOTE: postinstall_script is executed before postinstall_commands
